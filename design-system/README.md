# Design System Preview

This is a comprehensive preview page for the Larkon Admin Dashboard Design System.

## Files

- `preview.php` - Main preview page (HTML/PHP)
- `preview.css` - Complete CSS with all design tokens
- `preview.js` - JavaScript for interactivity
- `db-config.php` - Database configuration (optional)

## Running Locally

### Prerequisites
- XAMPP installed and running
- Apache web server started
- PHP enabled

### Steps

1. **Start XAMPP Services**
   - Open XAMPP Control Panel
   - Start Apache service
   - (Optional) Start MySQL if you want to use database features

2. **Access the Preview Page**
   - Open your browser
   - Navigate to: `http://localhost/cabinetry-dev/design-system/preview.php`
   - Or: `http://localhost/design-system/preview.php` (if you're in the design-system directory)

3. **View the Design System**
   - The page showcases all components from the design system
   - Includes: colors, typography, buttons, inputs, badges, alerts, tables, navigation, and more
   - All components are interactive
