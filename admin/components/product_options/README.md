# Product Options Component

An advanced, highly customizable product options component for creating dynamic product options for custom manufacturing (cabinetry, kitchens, wardrobes, etc.).

## Features

- **Extensible Datatype System**: Plugin-based architecture supporting 12+ built-in datatypes
- **Custom Query Builder**: Visual SQL query builder for database-driven options
- **Conditional Logic Engine**: Rule-based system for show/hide, filtering, dependencies
- **Advanced Pricing**: Formula-based pricing with per-option enable/disable
- **Visual Builder**: Drag-and-drop interface with live preview
- **Template System**: Save and reuse option configurations
- **Flexible Database Integration**: Custom queries on any table with parameter placeholders

## Installation

See [INSTALL.md](INSTALL.md) for detailed installation instructions.

## Quick Start

1. Navigate to `/admin/components/product_options/install.php`
2. Review auto-detected settings
3. Click "Install Product Options Component"
4. Access the dashboard at `/admin/components/product_options/admin/index.php`
